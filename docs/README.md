# Zenvik Documentation Index

## Core Docs
`docs/core/` controls brand, AI workflow rules, and production safety.

- `BRAND-GUIDE.md`: Zenvik colors, brand feel, and ecosystem visual rules.
- `CODEX-RULES.md`: permanent AI/Codex workflow rules.
- `PRODUCTION-RULES.md`: deployment, production, and safety constraints.

## Main-Site Docs
`docs/main-site/` controls the public website structure and visual system for `zenviktechnologies.com`.

- `FRONTEND-ARCHITECTURE.md`: main-site stack, responsibilities, and frontend structure.
- `HOMEPAGE-STRATEGY.md`: approved homepage positioning, service balance, and visual rhythm.
- `SERVICES-STRUCTURE.md`: main-site service categories and presentation rules.
- `UI-UX-SYSTEM.md`: visual design, interaction, layout, and portal matching principles.
- `WEBSITE-STRUCTURE.md`: page ownership and boundaries between main-site and portal.

## Portal Docs
`docs/portal/` controls the WHMCS portal strategy for `portal.zenviktechnologies.com`.

- `WHMCS-ARCHITECTURE.md`: WHMCS template rules, portal boundaries, and safety constraints.

## Agent Rules
`AGENTS.md` controls AI/Codex behavior in this repository. Read it before making changes, especially before editing main-site or portal files.
