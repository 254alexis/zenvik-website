# Master Codex Development Rules

## Objective
Build a production-grade enterprise ICT and hosting website for Zenvik Technologies.

## Technology Direction
Preferred stack:
- React
- Vite
- Tailwind CSS

Final output must support:
- static build export
- cPanel hosting deployment

## Critical Requirements
- Clean architecture
- Reusable components
- Mobile responsive
- SEO optimized
- Fast loading
- Professional animations
- Accessible UI
- Production-ready code

## Design Rules
Use:
- premium SaaS styling
- enterprise layout structure
- modern typography
- soft shadows
- gradients carefully
- clean spacing
- consistent card systems

Avoid:
- cluttered layouts
- oversized animations
- template-like appearance
- excessive colors
- inconsistent spacing

## Brand Colors
- Primary Blue: #043a7e
- Gold Accent: #7a6200
- CTA Orange: #dfa408

## Component Requirements
Every section should be:
- modular
- reusable
- scalable
- responsive

## Animation Rules
Use:
- Framer Motion where necessary
- lightweight transitions
- scroll reveal animations
- hover interactions

Animations must:
- remain performant
- support mobile devices
- not block rendering

## Homepage Priority
Priority order:
1. Hosting & Cloud
2. Software Development
3. Website Development
4. Marketing & Branding
5. ICT Services

## Code Quality
- Use clean naming conventions
- Avoid duplicate code
- Use scalable folder structure
- Separate sections into components

## WHMCS Compatibility
The visual system created for the main website must later be reusable for:
- WHMCS header
- WHMCS footer
- WHMCS buttons
- WHMCS cards
- WHMCS client area styling

## Deployment
Final build must support:
- GitHub workflow
- cPanel deployment
- manual upload builds
- future scalability
